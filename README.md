# NASA Turbofan Engine RUL Prediction

## Overview

This project predicts Remaining Useful Life (RUL) of turbofan engines using the NASA CMAPSS FD001 dataset.

## Models

- Random Forest Regressor
- XGBoost Regressor
- LightGBM Regressor

## Features

- Sensor Measurements
- Operational Settings
- Engine Cycle Information

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- LightGBM
- Matplotlib
- Seaborn
